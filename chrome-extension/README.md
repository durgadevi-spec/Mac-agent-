# Knockturn Monitor Link - Chrome Extension

This extension integrates Google Chrome tab/activity tracking with the Knockturn Employee Productivity Agent.

## Installation Instructions

1. Open **Google Chrome**.
2. Navigate to `chrome://extensions/` by typing it in the address bar.
3. Enable **Developer mode** using the toggle switch in the top-right corner.
4. Click the **Load unpacked** button in the top-left corner.
5. Select the `chrome-extension` directory (located at `c:\newaget\newagent-main\chrome-extension\`).
6. The extension is now installed! It will run in the background and automatically sync your active tab info to the agent whenever it is running.
